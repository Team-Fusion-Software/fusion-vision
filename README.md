# fusion-vision

Computer vision and perception for the Fusion drone platform.

## How to contribute

We welcome contributions! All changes are made through Pull Requests that a maintainer reviews and approves before merging — there are no direct pushes to `main`.

Before opening a PR, please read [CONTRIBUTING.md](CONTRIBUTING.md). In short: clone the repo, create a feature branch, push it, and open a Pull Request into `main`, then wait for maintainer review and approval.
